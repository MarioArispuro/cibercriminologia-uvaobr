¿No entiendes un término técnico? Aquí lo explicamos con ejemplos relacionados con tu carrera:

🔐 Firewall
Un muro que bloquea accesos no autorizados a una red.
Ej: Como un guardia en la entrada de una comandancia.
🦠 Malware
Software malicioso (virus, troyanos, etc.).
Ej: Un USB infectado que borra expedientes digitales.
🎣 Phishing
Engaño por correo para robar contraseñas.
Ej: Un mensaje falso que dice ser del C4 y pide tu usuario.
🔒 Criptografía
Técnica para cifrar información.
Ej: WhatsApp usa criptografía de extremo a extremo.
🕵️‍♂️ Ethical Hacking
Hackeo autorizado para encontrar fallos.
Ej: Un experto prueba la seguridad de una base de datos policial.
💾 Backups
Copias de seguridad.
Ej: Si el sistema de denuncias se cae, se restaura desde un respaldo.
📡 Ciberespacio
Entorno digital donde ocurren las comunicaciones.
Ej: Las redes sociales, correos, cámaras IP.
🧩 Vulnerabilidad
Debilidad en un sistema que puede ser explotada.
Ej: Un celular sin actualizaciones que permite espionaje.
🚨 Incidente de seguridad
Cualquier evento que afecta la confidencialidad, integridad o disponibilidad.
Ej: Un agente pierde su laptop con datos sensibles.
🛡️ Autenticación en dos pasos
Protección extra: contraseña + código por SMS o app.
Ej: Acceder al sistema de justicia con huella y token.

💡 Consejo: Usa este glosario durante tus investigaciones digitales. Imprímelo o guárdalo como referencia rápida.
